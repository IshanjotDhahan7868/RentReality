# Stack Decisions (V1)

- Frontend: Next.js 14, TypeScript, TailwindCSS
- Backend: Next.js API routes
- Database: Postgres on Supabase
- Data ingestion: Python scripts (run locally then later from a server/cron)
- AI: OpenAI API (GPT-4.x / GPT-5.1 Thinking when available)
- Hosting: Vercel for frontend + API; Supabase handles DB hosting
