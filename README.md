# RentReality

RentReality is a web app that solves housing affordability confusion for young renters and first-time buyers in Canada.

## V1 Goals

- Affordability Check tool (rent vs income with local data)
- "Can I Buy?" calculator for first-time buyers
- City & Data Explorer to visualize rent, income, and affordability over time
- AI-powered explanations for charts and personal scenarios

## Stack (planned)

- Next.js 14 + TypeScript + TailwindCSS
- Postgres (Supabase/Railway)
- Python ingestion scripts (CMHC, StatCan, BoC)
- OpenAI API for explanations
